How to Use

Step 1 — Get a Personal Access Token
1

Open Figma Account Settings

Click your avatar in the top-left of Figma, then choose Settings.

Avatar / Name › Settings
2

Navigate to Personal Access Tokens

In Settings, scroll to the Security section and click Personal access tokens.

Settings › Security › Personal access tokens
3

Generate a new token

Click Generate new token. Give it any name (e.g. ai-export). Under Scopes, enable File content → Read only. Copy the token immediately — it is only shown once.

warning

Copy it before closing the dialog — Figma won't show the token again after you navigate away.

Step 2 — Get Your Figma URL
A

Whole file URL

Copy the URL from your browser's address bar when the file is open. You'll pick a specific frame after connecting.

https://www.figma.com/design/AbCdEfGh1234/My-Form-Design
B

Link to a specific frame (recommended)

In Figma, right-click any frame on the canvas → Copy/Paste as → Copy link to selection. This auto-selects that frame.

Right-click frame › Copy/Paste as › Copy link to selection
https://www.figma.com/design/AbCdEfGh1234/My-Form?node-id=123-456
Step 3 — Select Frame & Export
1

Pick a page and frame

After connecting, use the dropdown to switch pages. Click any frame card to select it. Thumbnail previews load automatically in the background.

2

Download and open in Illustrator

Download the .SVG or .AI file. In Illustrator: File → Open. All shapes and text will be fully editable. If prompted about units, choose px.

tips_and_updates

.SVG is recommended — Illustrator treats it as a native first-class format. The .AI wrapper is useful if your pipeline specifically requires an .ai extension.

Privacy & Security
✓

Your token never leaves your browser

All API requests go directly from your browser tab to api.figma.com. No server is involved. The token is held only in memory and clears the moment you close or refresh this tab.
