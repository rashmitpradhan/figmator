How to Use Figmator

Step 1 — Personal Access Token
1

In Figma, click your avatar → Settings → Security → Personal access tokens.

2

Click Generate new token. Under Scopes, enable File content → Read only.

3

Copy the token immediately — Figma only shows it once.

open_in_new

Official Guide — Personal Access Tokens

Step-by-step on the Figma help center ↗
Step 2 — Figma File URL
A

Whole file: Copy the URL from your browser's address bar while the file is open.

B

Specific frame: Right-click any frame on the canvas → Copy/Paste as → Copy link to selection. Figmator will auto-select that frame.

api

Figma REST API — Images Endpoint

How Figmator exports frames as SVG ↗
Step 3 — Select & Export
1

Click frames to select them — multiple selection supported. Switch pages with the dropdown.

2

Choose SVG or AI output format, then click Export.

3

In Illustrator: File → Open the downloaded file. All paths and text are editable. Choose px if prompted for units.

tips_and_updates

SVG is recommended for Illustrator — it's treated as a native first-class format. Use .ai if your workflow specifically requires that extension.
