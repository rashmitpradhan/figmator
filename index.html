How to Use Figmator

Step 1 — Personal Access Token
1

In Figma, click your avatar → Settings → Security → Personal access tokens.

2

Click Generate new token. Name it anything. Under Scopes, enable File content → Read only.

3

Copy the token immediately — Figma only shows it once. Paste it into Figmator.

Step 2 — Figma File URL
A

Whole file: Copy the URL from your browser's address bar while the file is open. Figmator will list all frames from any page.

B

Specific frame (recommended): Right-click any frame on the canvas → Copy/Paste as → Copy link to selection. Figmator will auto-select that frame.

Step 3 — Select Frames & Export
1

After connecting, switch pages using the dropdown. Click frames to select them — you can select multiple at once.

2

Single frame: Downloads as .svg + .ai. Multiple frames: Downloads as a .zip containing all SVG files.

3

In Illustrator: File → Open the downloaded file. All paths, shapes, and text are fully editable. If prompted about units, choose px.

tips_and_updates

.SVG is recommended — Illustrator treats it as a native format. The .ai wrapper is useful if your workflow requires an .ai file extension specifically.
